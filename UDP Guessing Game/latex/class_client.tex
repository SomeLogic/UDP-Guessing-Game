\doxysubsubsection{\texorpdfstring{PlayGame()}{PlayGame()}}
{\footnotesize\ttfamily void Client\+::\+Play\+Game (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Manages gameplay phase. 

Receives game range, then in loop received packet is checked for operation and answer values. If operation equals \textquotesingle{}game\+End\textquotesingle{} game is finished and program ends its run. Otherwise it is checked if answer was wrong, out of range or if it was given during other client\textquotesingle{}s turn. In every loop repeat user enters new number from game range and counter is being updated. \begin{DoxySeeAlso}{See also}
